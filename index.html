This site is deployed automatically using GitHub, Jenkins and XAMPP. Every change pushed to GitHub is built and copied to this server by Jenkins.

About

My name is Burutolu Roy. This project shows a basic DevOps pipeline: code is stored in GitHub, Jenkins pulls the repository, and deploys the files to an Apache server running on XAMPP.

The page you are viewing is a single-page layout with sections for home, about and contact. Navigation links simply scroll to each section.
