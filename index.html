Welcome

This site is deployed automatically using GitHub, Jenkins and XAMPP. Every change I push to GitHub is built and copied to this server by Jenkins.

Version 2.1

About

My name is Burutolu Roy. This project demonstrates a simple CI/CD (Continuous Integration / Continuous Delivery) pipeline:

  • Code is stored in a repository on GitHub.
  • Jenkins pulls the repository when changes are pushed.
  • Files are deployed to an Apache web server running on XAMPP.

The page you are viewing is a one-page layout with sections for home, about and contact. The navigation links at the top jump smoothly to each section.
